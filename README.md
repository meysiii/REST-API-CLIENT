# ğŸ“„ Project Description

The Java REST API Client is a console-based application that consumes a public REST API to fetch real-time weather information for a given city.

The application sends an HTTP GET request to the OpenWeatherMap API, receives the response in JSON format, parses the data, and displays it in a clean, structured, and user-friendly manner.

Unlike static or offline programs, this project demonstrates how Java applications can dynamically retrieve live data from external services. It showcases the fundamental working of REST APIs without using complex frameworks, making it simple, beginner-friendly, and easy to understand.
